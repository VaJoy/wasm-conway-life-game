# wasm-conway-life-game
 AssemblyScript 简易版康威生命游戏
 
[点击查看康威生命游戏规则](https://baike.baidu.com/item/%E5%BA%B7%E5%A8%81%E7%94%9F%E5%91%BD%E6%B8%B8%E6%88%8F/22668799)

## 执行步骤

### 一、安装模块：

```
pnpm i
```


### 二、启动 server：

```
npm run server
```


### 三、构建 wasm：

```
npm run asbuild
```

完成后访问页面 `http://localhost:3000/index.html` 即可。
